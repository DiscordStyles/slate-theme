What is Slate?

Customizable

Slate is one of the most customizable themes out there. You can edit almost anything about the appearance from fonts to backgrounds. We also offer pre-made Palettes.

Bb

Consistent

Slate's current compiled base file is over 12,000 lines long, and touches almost every part of the app. It's design is completely consistent from the colors to font scaling.

Optimized

Animations on Slate are almost non-existent, and are very minimal when utilized. Slate avoids the use of properties such as translate() to position elements and global selectors when possible.

Cross-compatible

Slate is developed for both BD and Powercord, with Windows, Linux, and macOS support in mind. We also plan to include web versions and support for other mods, such as Vizality and EnhancedDiscord.
